# Component Audits

Add one file per component using the naming convention:
- box.json
- button.json
- form-field.json
- text-input.json

Each file should match the schema in [schemas/prop-audit-schema.json](../schemas/prop-audit-schema.json).

Use this table to track status:

| Component | Status | Notes |
| --- | --- | --- |
| Box | complete | JSON in components/box.json |
| Button | complete | JSON in components/button.json |
| FormField | complete | JSON in components/form-field.json |
| TextInput | complete | JSON in components/text-input.json |
