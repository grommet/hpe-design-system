## Use cases

### Providing inline contextual help

A Popover may be triggered through an icon-only button to provide the user with helpful contextual info in a non-intrusive manner.

## Anatomy

| Label  | Region                  | Purpose                               | Required | Notes                                               |
| :----: | ----------------------- | ------------------------------------- | :------: | --------------------------------------------------- |
| **1**  | **Title**               | Identifies the Popover.               | Required | Implemented as Heading level 3.                     |
| **2**  | **Close Button**        | Closes the Popover.                   | Required | --                                                  |
| **3**  | **Body**                | Provides contextual information.      | Required | May include other elements that are not text.       |
| **4**  | **Footer**              | Contains additional Popover actions.  | Optional | May contain anchors.                                |

## Accessibility
Be sure to include an a11yTitle on:
- Any triggers without a label such as an icon-only button. 
- The Close button describing the contents of the Popover
