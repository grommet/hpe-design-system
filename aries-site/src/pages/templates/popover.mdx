## Use cases

### Providing inline contextual help

A Popover may be triggered through an icon-only button to provide the user with helpful contextual info in a non-intrusive manner.

| Label  | Region                  | Purpose                               | Required | Notes                                               |
| :----: | ----------------------- | ------------------------------------- | :------: | --------------------------------------------------- |
| **1**  | **Title**               | Identifies the Popover.               | Optional |                                                     |
| **2**  | **Close Button**        | Closes the Popover.                   | Required | --                                                  |
| **3**  | **Body**                | Provides contextual information.      | Required | May include other elements that are not text.       |
| **4**  | **Footer**              | Contains additional Popover actions.  | Optional | May contain anchors.                                |

## Content
Popovers should be:
- Relevant - The Popover should relate to its trigger and provide additional contextual information.
- The content should not contain crucial information or actions required to complete a task.
- Short -  Messages should be simple, too the point, and small to avoid overloading the user and taking up too much screen real estate. Follow the voice and tone guidelines for writing crisp and effective content.

It is recommended that Popovers:
- Use Paragraph/Small and Anchor/Small to save real estate.
- Avoid using titles if it provides no additional context.

## Accessibility
Be sure to include an a11yTitle on:
- Any triggers without a label such as an icon-only button. 
- The Close button describing the contents of the Popover.

| Key                   | Interaction                                                                                                                                          |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Tab**               | Highlights interactive elements. The focus will remain inside the Popover until exited. Once exited, the focus will return to the Popover’s trigger. |
| **Esc**               | Used to exit the Popover at any time.                                                                                                                    |                                                                                                    |
