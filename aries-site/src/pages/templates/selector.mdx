## Anatomy


| Label | Region                  | Purpose                                                      | Required | Notes                                                   |
| :---: | --------------------    | ------------------------------------------------------------ | :------: | ------------------------------------------------------- |
| **1** | **Title**               | Heading providing context to the options available.          | Optional | Titles should use a Heading of the correct semantic heading level based on where the selector is in the document. |
| **2** | **Description**         | Brief explanation or additional information about the choices.| Optional |    --                                                  |
| **3** | **Counter of Items**    | Displays number of items available for selection.            | Optional | -- |
| **4** | **Icon**                | Visual aid to represent the category or type of selection.   | Optional | a11yTitle required. |
| **5** | **Selector Indicator**  | Visual cue (such as a checkmark or highlight) to denote selected items. | Optional | aria-label required. |

## Types

### Single select

With single selection, only one option can be selected at a time. This is the default behavior.


### Multiple select

With multiple selection, multiple options can be selected on at the same time.
