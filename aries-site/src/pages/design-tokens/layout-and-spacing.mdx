## Versioning

### Current version: v7

The v7 scale introduced intermediate sizes and renamed tokens for improved granularity and consistency. Built from dual base units (`24px` primary and `16px` supplementary), the scale follows a proportional doubling/halving pattern.

Primary ladder (24×2ⁿ) yields 24, 48, 96, 192, 384, 768, 1536. Supplementary ladder (16×2ⁿ) yields 16, 32, 64, 128, 256, 512, 1024. Interleaving creates a smooth progression without arbitrary values.

**Note:** Version numbers refer to [Grommet HPE theme versions](https://github.com/grommet/grommet-theme-hpe/releases).

**Key v7 improvements:**
- Added 5 new intermediate sizes for finer control
- Renamed tokens for clearer size progression (e.g., `xxsmall` → `5xsmall`)
- Simplified border sizes, removed overly thick options
- Added `none` and `default` values where appropriate

See [T-shirt sizing foundation](/foundation/tshirt-sizing) for conceptual understanding of the scale principles.

### About v6
Theme v6 used a single 24px base unit with a halving/doubling ladder for spacing (3 → 6 → 12 → 24 → 48 → 96), parallel container widths (48 → 96 → 192 → 384 → 768 → 1152 → 1536), four radius steps (2, 4, 8, 16), and five border widths (1, 2, 4, 12, 24), providing a compact, consistent token set.

### Migration guide

Upgrading from v6 to v7 requires updating token names and reviewing sizing choices where intermediate sizes now exist.

**Migration strategy:**
1. **Spacing and containers**: Most v6 tokens map to renamed v7 equivalents (e.g., `small` → `xsmall`, `large` → `xlarge`)
2. **Consider intermediate sizes**: New options like `small`, `large`, `xxsmall`, `xxlarge` may better fit your needs
3. **Border widths**: Replace removed `large` (12px) and `xlarge` (24px) with smaller alternatives
4. **Test responsive behavior**: The scale now steps down more granularly at breakpoints

**Automated migration:**
Use the codemods provided in the migration guide to update token references automatically.

See the complete [v6 to v7 migration guide](https://github.com/grommet/hpe-design-system/blob/master/packages/codemods/migration_guides/MIGRATE_THEME_V6_TO_V7.md) for detailed instructions, code examples, and automated migration tools.
