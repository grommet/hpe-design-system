## Guidance

- DateInput can be used to display past, present, or future dates and times.
- DateInput is an intuitive way for users to input a date or time by providing them with a familiar calendar interface.

### When using Date Input, you should:

- Use a label when appropriate
* Correctly format to reduce mistakes such as having the caller specify
  the appropriate date format for the locale the user is in. 
- For example, in the U.S. dates are formatted as ‘mm/dd/yyyy’ whereas
 in other countries, such as Europe it might be ‘dd/mm/yyyy’.

### Usage
* Use the date input component when you want the user to assign past,
  present or future date(s).

## Variants
- Date Input - This date input can used to pick one day or a range of dates,
depending on your use case. Concise labels should be included on this variant
to let the user know what to do.

### Single Date

end date is needed from the user. Users will need to use the calendar
to select the range of dates.
