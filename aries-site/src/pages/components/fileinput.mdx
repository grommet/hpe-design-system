## Guidance

FileInput is a specialized input that provides a clean interface for selecting files.
FileInput can be used when a user needs to upload one or more files.

### Usage

1. Upload a file or files by click on the fileInput that prompts a file selection.
2. Drag and Drop selected files directly into the fileInput.
3. Label the fileInput so the user knows what file/files needs to be uploaded

### Accessibility

The FileInput can be used with both the `Space` and `Enter` button to achieve the action.
FileInput should have clear label that describes the type of files that are needed.

## Variants

FileInputs can be used as a single file upload or multiple files.
