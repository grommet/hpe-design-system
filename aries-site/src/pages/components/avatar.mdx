## Anatomy

Avatars can be displayed as an image, initials, or a logo. They come in three different sizes:

- `small`: 24x24px
- `medium`: 48x48px
- `large`: 96x96px

The container of an Avatar can be circular or a rounded square based on the use case.

- Use a circular Avatar when representing a person by an image or their initials.
- Use a rounded square Avatar when representing a company’s or entity’s logo. This can be achieved by passing `round='medium'`
- Always round the edges of a square container when using an avatar.

## When to use Avatars

Typically, Avatars are used mainly as an easy identifier for users and are always paired with additional information for better clarity. If possible, prioritize using an image or logo over initials.

## Variants

### DataTable

When using Avatars in a DataTable, always provide the user’s or company’s name or email alongside it. If an image is not available provide an Avatar with initials instead. Use small-sized avatars in DataTables.

### Card

Both Avatars and Icons can be used within Cards. Reminder that...

- Avatars represent a person, company, or entity.
- Icons represent objects, actions, programs, and other communicative symbols.

Use medium-sized avatars within cards.
