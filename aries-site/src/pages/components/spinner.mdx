## Guidance

Loading Spinners are used to notify the user that an action is being processed. 
This reassures the user that an action is occuring and will be provided shortly. 

Loading spinners can be made smaller or larger depending on the context in which the spinner is being used.

For fast processes that take less than 300ms, consider a spinner.

### When to use Spinner

Use a spinner when an action requires a short time to process for example: (downloadnig, uploading, processing, etc).
When loading more information into a table or list a spinner should be used.

### Accessibility

In order for a spinner to be accessible we need to provide a [message](https://v2.grommet.io/spinner#message) that will be announced for screen readers.
