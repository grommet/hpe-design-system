## Guidance

Tooltips can be added to Button by Grommet [tip props](https://v2.grommet.io/button#tip).
Tooltips will display information upon hover and focus.
Tooltips provide messages that give the user more information about an element.

### Usage

1. Don't use Tooltips for information that is vital to task completion.
2. Provide a brief and helpful content inside the Tooltip.
3. Support both mouse and keyboard hover.
4. Use Tooltip consistently throughout your site.
5. Tooltip should never contain interactive components.

Make sure to test Tooltip positioning to ensure that the content does not block other information pertinent to the user's goal.

### Accessibility

Tooltips should support both mouse hover and keyboard focus.
Do not use tooltips for information that is vital for the user to complete the task.

### Truncation with tooltip

To display the full text when truncation occurs, hover over the truncated text and a tooltip will be shown with the full text.

Upon hovering, Tooltip presents additional information about a specific element in a small container. The Tooltip will remain visible as long as the user is hovering over the element.

## Variants

There are a few different ways that you can use a Tooltip.

### Tooltip with icons

An icon without a label should be provided a Tooltip to remove any ambiguity around an icon's meaning. An example can be found in a demonstration of [Persistent Navigation](/templates/persistent-navigation#minimal-sidebar), where a sidebar contains icon-only buttons.
