## Guidance

Tooltips will display information upon hover and focus.
Tooltips provide messages that give the user more information about an element.

### Usage

1. Don't useTooltips for information that is vital to task completion.
2. Provide a brief and helpful content inside the Tooltip.
3. Support both mouse and keyboard hover.
4. Use Tooltip arrows when multiple elements are nearby.
5. Use Tooltip consistently throughout your site.

Make sure to test Tooltip positioning to ensure that the content does not block other information pertinent to the user's goal.

### Accessibility

Tooltips should support both mouse and keyboard hover.
Do not useTooltips for information that is vital for the user to complete the task.

## Variants

There are a few different ways that you can use a Tooltip. 

### Tooltip with icons

An icon without a label should be provided a Tooltip. Some of the icons have some level of ambiguity which is why this is recommended. This would be also a use case for icons in any sidebar, an example can be found in the [Persistent Navigation](/templates/persistent-navigation#minimal-sidebar)
