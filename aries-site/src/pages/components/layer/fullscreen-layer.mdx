## Anatomy

This diagram highlights details specific to fullscreen layers. See [general layer anatomy](/components/layer#anatomy) for specifications applying to all layers.

Fullscreen layers fill the entire page. Therefore, there is no rounding or elevation visible.

<Example plain>
  <FullscreenLayerAnatomy />
</Example>

| Label | Region             | Purpose                                                           | Required?                                  | Notes                                                                                                                               |
| ----- | ------------------ | ----------------------------------------------------------------- | ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Close button**   | Closes the layer.                                                 | Required on layers with no footer actions. | Use "FormClose" icon.                                                                                                               |
| **2** | **Footer actions** | Provides actions for user to accept layer consequences or cancel. | Required on actionable layers.             | The actions should be right-aligned with primary action on the right. One action should be a "Cancel" button that closes the layer. |

See [layer closing behavior](/components/layer#layer-closing-behavior) for details.
