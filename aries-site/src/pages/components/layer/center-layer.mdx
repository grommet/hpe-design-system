## Anatomy

This diagram highlights details specific to center layers. See [general layer anatomy](/components/layer#anatomy) for specs that apply to all layers.

<Example plain>
  <CenterLayerAnatomy />
</Example>

| Label | Region             | Required?                                  | Purpose                                                           | Notes                                                                                                                               |
| ----- | ------------------ | ------------------------------------------ | ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Close button**   | Required on layers with no footer actions. | Closes the layer.                                                 | Use "FormClose" icon.                                                                                                               |
| **2** | **Footer actions** | Required on actionable layers.             | Provides actions for user to accept layer consequences or cancel. | The actions should be right-aligned with primary action on the right. One action should be a "Cancel" button that closes the layer. |

See [layer closing behavior](/components/layer#actionable) for details.
