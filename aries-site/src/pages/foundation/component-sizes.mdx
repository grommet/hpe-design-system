T-shirt size tokens define component dimensions via a component's size
prop. Sizes are most often available in `small`, `medium`, and `large`, however
may extend into `xxsmall` or `3xlarge` ranges for some components.

Example components supporting t-shirt sizes include:

- Typography - [Heading](/foundation/typography#heading), [Paragraph](/foundation/typography#paragraph), [Text](/foundation/typography#text)
- Controls - [Anchor](/components/anchor), [Button](/components/button), [Pagination](/components/pagination)
- Visualizations - [Avatar](/components/avatar), [Spinner](/components/spinner), [Tag](/components/tag)

## Default sizes

Components exposing a `size` property default to `medium` when unspecified.

In the button example below, “Default” and size=”medium” produce the equivalent rendering:

```jsx
<Button label="Default" primary />
<Button label="Medium" primary size="medium" />
```

Omitting the `size` prop when `medium` is appropriate. Omission reduces verbosity without altering appearance.
