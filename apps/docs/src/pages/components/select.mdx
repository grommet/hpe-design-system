## Accessibility

Always use Select in conjunction with FormField to ensure the label is properly associated with the input for assistive technologies.

To properly associate FormField `label` with Grommet's Select component, ensure the Select has an `id` and the FormField has an `htmlFor` with the format of `${id}__input`.

### Keyboard interaction

| Key | Action |
| --- | --- |
| Space or Enter | Toggles the open/closed state of the drop-down list. |
| Arrow keys | Navigates through options when the list is open. |
| Tab | Moves focus to the next focusable element. |
| Escape | Closes the drop-down without changing the selected value. |
