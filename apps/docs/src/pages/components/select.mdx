## Accessibility

Always use Select in conjunction with FormField to promote best practices in accessibility by ensuring user inputs are accompanied by labels.

To properly associate FormField `label` with Grommet's Select component, ensure the Select has an `id` and the FormField has an `htmlFor` with the format of `${id}__input`.

### Keyboard interaction

| Key | Action |
| --- | --- |
| Space or Enter | Toggles the open/closed state of the Select drop-down. |
| Tab | Moves focus to the next element. |
| Arrow keys | Navigates through the options list when open. |
