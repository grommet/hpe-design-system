| :---: | ------------- | ------------------------------------------------------------ | :------: | ----------------------------------------------------------- |
| **1** | **Container** | Organizes children, how they appear and spaced.              | Required | --                                                          |
| **2** | **Label**     | Describes what the button will do when selected.             | Optional | If no label is shown, icon is required.                     |
| **3** | **Icon**      | Used to visually represent text label with or without label. | Optional | a11yTitle required. If no icon is shown, label is required. |

## Types

### Single select

With single selection, where only one button can be toggled on at a time. This is the default behavior.

### Multiple select

With multiple selection, multiple buttons can be toggled on at the same time.

## Accessibility

In icon-only cases, ensure each icon has a clear `aria-label`.

| Key                   | Interaction                                                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| **Tab**               | If nothing is toggled on, focuses on the first button in the group. Otherwise, focuses on the first button that is toggled on. |
| **Left/Right Arrows** | Moves to other available buttons.                                                                                              |
| **Space/Enter**       | Selects button with focus.                                                                                                     |
